# Aula 01: Introdução ao HTML - Estrutura e Elementos

## Tópicos Estudados

Nesta aula, aprendemos os conceitos fundamentais da linguagem de marcação HTML.

- **O que é HTML**: Linguagem de Marcação de Hipertexto, interpretada pelo navegador para renderizar páginas web.
- **Tags**: Estruturas que marcam o conteúdo, como `<tag>` (abertura) e `</tag>` (fechamento).
- **Estrutura Básica**: Todo documento HTML possui uma estrutura inicial com as tags `<html>`, `<head>` (para metadados como o título da página) e `<body>` (para o conteúdo visível).
- **Elementos de Formulário**:
  - `<h1>` a `<h6>`: Tags para títulos e subtítulos.
  - `<input>`: Elemento versátil para criar campos de formulário. Vimos vários tipos, como `text`, `password`, `checkbox`, `radio`, `date`, `color`, `file`, etc.
  - `<textarea>`: Para caixas de texto com múltiplas linhas.
  - `<select>` e `<option>`: Para criar listas de seleção (dropdown).
  - `<button>`: Para criar botões.
- **Atributos**: Características adicionais para as tags, como `size`, `maxlength`, e `placeholder`.
- **Tabelas**: Como estruturar dados usando as tags `<table>`, `<tr>` (linha), `<td>` (célula), `<thead>` e `<tbody>`.

O exercício prático desta aula consistiu em criar um formulário com pelo menos 10 campos diferentes para validar o aprendizado sobre os elementos de input.
