# Aula 03: Transição de Páginas na Web

## Resumo

Nesta aula, focamos em como construir um projeto web com múltiplas páginas e como navegar entre elas.

### Tópicos Principais:

- **Organização de Projetos:** Aprendemos a estruturar um projeto web de forma organizada, separando arquivos por tipo em pastas dedicadas (`paginas/`, `css/`, `js/`).
- **Hiperlinks com a Tag `<a>`:** Usamos a tag `<a>` e seu atributo `href` para criar links que permitem a navegação entre as diferentes páginas HTML do projeto.
- **Caminhos Relativos:** Praticamos o uso de caminhos relativos para acessar arquivos em diferentes diretórios. Por exemplo:
  - `paginas/cadastro.html`: Para acessar um arquivo em uma subpasta.
  - `../index.html`: Para "voltar" um nível de diretório e acessar um arquivo.

O projeto prático consistiu em criar um sistema simples com 3 páginas (Login, Cadastro e Principal), todas interligadas por links de navegação.
