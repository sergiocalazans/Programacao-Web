## Introducao

Este repositorio contem todos os exercicios, projetos e anotacoes desenvolvidos durante a disciplina de **Programacao Web**, ofertada pela Escola Politecnica da PUCPR no segundo semestre de 2025.

A disciplina, de natureza pratica, introduz o desenvolvimento de sistemas web interativos e responsivos, com comunicacao assincrona entre cliente-servidor e persistencia de dados. O objetivo e capacitar o estudante a construir websites modernos, empregando boas praticas e as tecnologias essenciais do front-end.

---

## Conteudos Abordados

Ao longo do semestre, os seguintes temas de estudo (Study Topics) foram explorados:

- **TE1:** Construcao de elementos estaticos de paginas web (HTML).
- **TE2:** Formatacao de paginas web (CSS).
- **TE3:** Programacao em JavaScript para interatividade.
- **TE4:** Comunicacao cliente e servidor.
- **TE5:** Persistencia em banco de dados relacional.
- **TE6:** Boas praticas de programacao web.
