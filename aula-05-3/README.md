# Aula 05: Introdução ao JavaScript

## Resumo

Nesta aula, demos o primeiro passo para adicionar interatividade e lógica às nossas páginas web, conhecendo a linguagem de programação JavaScript.

### Tópicos Principais:

- **O que é JavaScript:** Entendemos o papel do JS como a terceira camada do front-end (HTML para estrutura, CSS para estilo, JS para comportamento).
- **Inclusão de Scripts:** Aprendemos a vincular um arquivo `.js` a uma página HTML usando a tag `<script>`, e a importância de colocá-la no final do `<body>` para evitar erros de carregamento.
- **Manipulação do DOM:** Usamos `document.getElementById()` para selecionar elementos HTML e interagir com eles (ler ou alterar seus valores).
- **Eventos:** Implementamos a interatividade com o evento `onclick` em botões para executar funções JavaScript.
- **Funções e Lógica:** Criamos funções para realizar cálculos (soma, subtração, etc.) e tomar decisões (usando `if/else`).
- **Conversão de Tipos (Type Casting):** Descobrimos a importância de converter valores de texto (string) para números usando `parseInt()` antes de realizar operações matemáticas.

Os três exercícios práticos nos guiaram desde a comparação de valores até a criação de uma calculadora funcional, aumentando a complexidade e solidificando os conceitos aprendidos.
