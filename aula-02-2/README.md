# Aula 02: Introdução ao CSS

## Resumo

Nesta aula, fomos introduzidos à segunda camada fundamental do front-end: o CSS (Cascading Style Sheets). O foco foi aprender como aplicar estilos para formatar e dar vida aos nossos documentos HTML.

### Tópicos Principais:

- **O que é CSS:** Entendemos que o CSS é uma linguagem de formatação usada para controlar a aparência de elementos HTML (cores, fontes, espaçamentos, etc.).
- **Formas de Aplicação:** Vimos as duas principais formas de aplicar CSS:
  1.  **CSS Interno:** Utilizando a tag `<style>` diretamente no `<head>` do arquivo HTML.
  2.  **CSS Externo (Melhor Prática):** Criando um arquivo separado (`.css`) e vinculando-o à página com a tag `<link rel="stylesheet" href="...">`.
- **Seletores de Classe:** Aprendemos a usar `class` como o principal método para conectar um estilo CSS a um ou mais elementos HTML.
- **Propriedades Básicas:** Praticamos com propriedades essenciais como `background-color`, `color`, `border`, `padding`, e `font-family`.
- **Pseudo-classes:** Fomos introduzidos a seletores que aplicam estilos a um estado especial de um elemento, como:
  - `:hover`: Para aplicar um estilo quando o mouse está sobre o elemento.
  - `:nth-child()`: Para selecionar elementos com base em sua posição (ex: criar "listas zebradas" em tabelas, estilizando linhas pares e ímpares de forma diferente).

Os exercícios práticos focaram em replicar um formulário com um design específico e em estilizar uma tabela, aplicando todos os conceitos aprendidos.
