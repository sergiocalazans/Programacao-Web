# Aula 04: Construção de Cards com `<div>` e Flexbox

## Resumo

Esta aula foi focada em layout e design de componentes, introduzindo conceitos modernos de CSS para criar interfaces visuais.

### Tópicos Principais:

- **A Tag `<div>` como Container:** Entendemos como a `<div>` funciona como um container genérico para agrupar elementos, sendo a base para a criação de componentes complexos.
- **Introdução ao Flexbox:** Tivemos o primeiro contato com o `display: flex`, um poderoso modelo de layout do CSS3 que permite alinhar, distribuir e organizar elementos dentro de um container de forma flexível e responsiva.
- **Construção de Cards:** Aplicamos os conceitos de `div` e Flexbox para construir dois "Cards" diferentes, que são componentes de UI muito comuns em aplicações web modernas.

Os exercícios consistiram em replicar o design de dois cards a partir de um layout pré-definido, organizando o conteúdo em HTML e aplicando o estilo em CSS.
